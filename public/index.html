Devices

View, register and remove devices

Devices

List devices

Query one or more registered devices

id:
string

Device ID (overrides other parameters)

tags:
TagInput

List of tag filters

parents:
string[]

List of IDs of parent devices that matching devices must have

bbox:
BBox

Geographic bounding box. Results outside this box will not be returned

searchTerm:
string

Free text search field to filter by device display name, IP address and/or MAC address

Example

Request Content-Types: application/json
Query
query devices($id: ID, $tags: [TagInput!], $parents: [ID!], $bbox: BBox, $searchTerm: String){
  devices(id: $id, tags: $tags, parents: $parents, bbox: $bbox, searchTerm: $searchTerm){
    devices{
      id
      tags
      parents
      displayName
      img
      iconType
      specs
      status
      mac
      probeIp
      nztpWorkflow
      ipmiAddress
      ipmiUsername
      hostIp
      telemetryUrl
      site
    }
    hits
  }
}
Variables
{
  "id": "string",
  "tags": [
    {
      "key": "string",
      "inc": "boolean"
    }
  ],
  "parents": [
    "string"
  ],
  "bbox": {
    "nw": {
      "lat": "number",
      "lng": "number"
    },
    "se": {
      "lat": "number",
      "lng": "number"
    }
  },
  "searchTerm": "string"
}
Try it now
200 OK

Successful operation

type
DevicesResult
Response Content-Types: application/json
Response Example (200 OK) 
{
  "data": {
    "devices": {
      "hits": "integer"
    }
  }
}
Devices

List nearby devices

Find devices within range of a reference device

device:
string

ID of the reference device

tags:
TagInput

List of tag filters

radius:
number

Radius in metres of a circular cap on the Earth's surface, centered on the position of the reference device. Results outside this cap will not be returned.

latency:
number

Maximum network latency with the reference device, in milliseconds

Example

Request Content-Types: application/json
Query
query nearbyDevices($device: ID!, $tags: [TagInput!], $radius: Float, $latency: Float){
  nearbyDevices(device: $device, tags: $tags, radius: $radius, latency: $latency){
    hits
  }
}
Variables
{
  "device": "string",
  "tags": [
    {
      "key": "string",
      "inc": "boolean"
    }
  ],
  "radius": "number",
  "latency": "number"
}
Try it now
200 OK

Successful operation

type

Response Content-Types: application/json
Response Example (200 OK) 
{
  "data": {
    "nearbyDevices": {
      "hits": "integer"
    }
  }
}
